# Memory Puzzle Game (Надоградена верзија)
Оваа апликација претставува 2D игра инспирирана од класичната Memory Puzzle игра, развиена со користење на Pygame библиотеката во Python. Играчот треба да пронајде парови од исти симболи (форма и боја) со отворање на полиња од таблата.

## Карактеристики
* Класична Memory Puzzle механика
* Табла со димензии 10 × 10
* Додадена нова форма (triangle)
* Додадени нови бои (magenta и lime)
* Hint опција за помош
* Score систем со серија (streak)
* Анимации за откривање и покривање полиња
* Почетна анимација и win анимација

## Дополнителни барања
### Промена на таблата
Таблата е зголемена на 10 × 10, со што се зголемува тежината на играта. Осигурано е дека бројот на полиња е парен за да може да се формираат парови.
### Нови форми и бои
  * Додадена е нова форма: triangle
  * Додадени се нови бои: magenta и lime
Ова овозможува поголема разновидност на комбинации.
### Hint функционалност
Имплементирано е копче Hint кое функционира на следниот начин:
  * Откако играчот ќе отвори едно поле
  * Може да кликне на Hint
  * Играта кратко го означува соодветниот пар
### Score систем
Имплементиран е систем на поени базиран на серија на точни погодувања:
  * 1 погодок → +1 поен
  * 2 по ред → +2 поени
  * 3 по ред → +4 поени
  * 4 по ред → +8 поени
Поените растат експоненцијално со формула:
```
score += 2^(streak - 1)
```
Ако играчот направи грешка: streak се ресетира на 0

## Логика на играта
Играчот отвора по две полиња:
* Ако се исти → остануваат откриени
* Ако не → повторно се покриваат
Играта завршува кога: сите полиња се откриени

## Структура на кодот
Главни функции:
* main() – иницијализација и главен loop
* getRandomizedBoard() – генерирање на таблата
* drawBoard() – цртање на таблата
* revealBoxesAnimation() / coverBoxesAnimation() – анимации
* showHint() – Hint логика
* hasWon() – проверка за победа
